God of Dice

Бот для поддержки настольной ролевой игры Savage Worlds (в основном Adventure Edition) в Discord.
Основные возможности


Быстрые проверки навыков и характеристик

!s8!s10+2!e12-1!d6

(s — с диким кубиком, e — без дикого кубика, d — обычный бросок без взрывов)

Бросок урона с поддержкой любых выражений

!2d6+1d8+2+d4-3!!3d6+d8+d6!

Полная поддержка взрывов, знаки ±, несколько кубиков

Система инициативы 

!f / !fight — начать новый бой

!di Имя-q-l-h Имя2 Имя3-i — раздать карты

Поддерживаемые черты:

-q — Quick (стремительность, вытягивает дополнительные карты пока не вытенит карту 6 или больше)

-l / -i — Level Headed / Improved Level Headed (2 или 3 карты, выбирается лучшая)

-h — Hesitant (медлительность, выбирается худшая из двух карт)


Управление раундами и персонажами

!rd — новый раунд

!rd + — сохранить всех персонажей в новом раунде

!rd + -Вася -Петя — сохранить всех, кроме указанных

!card Имя — вытянуть новую карту персонажу

!hold Имя / !hold -Имя — поставить/снять с ожидания

!drop Имя1 Имя2 — убрать персонажей из инициативы

Прочие утилиты

!drawcard [кол-во] — вытянуть обычные игральные карты (1–10)

!drawcarb — гарантированно вытянуть красного джокера (Теперь у каждого есть своя личная колода Джокеров))

!invite — ссылка-приглашение бота
